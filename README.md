## Getting Started

Download [latest release](https://github.com/justaCasualCoder/Frigatify/releases/tag/latest) and install it:
```bash
wget https://github.com/justaCasualCoder/Frigatify/releases/tag/latest
chmod +x frigatify
sudo mv frigatify /bin/
```
